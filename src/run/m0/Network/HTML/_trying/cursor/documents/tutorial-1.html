Pierwszy tutorial

W tym tutorialu nauczysz się podstaw korzystania z systemu.

Kroki

  1. Zainstaluj system
  2. Skonfiguruj podstawowe ustawienia
  3. Utwórz pierwszego użytkownika
  4. Przetestuj podstawowe funkcje

Wymagany czas

Około 30 minut

Wymagania wstępne

  • Node.js w wersji 14+
  • NPM lub Yarn
  • Przeglądarka internetowa

Krok 1: Instalacja

Zainstaluj system używając NPM:

npm install -g nasz-system

Krok 2: Konfiguracja

Uruchom system z domyślnymi ustawieniami:

nasz-system start

Krok 3: Pierwszy użytkownik

Przejdź do http://localhost:3000/admin i utwórz konto administratora.

Krok 4: Testowanie

Sprawdź czy system działa poprawnie:

  • Zaloguj się do panelu administracyjnego
  • Utwórz kilka testowych użytkowników
  • Przetestuj API endpoints
