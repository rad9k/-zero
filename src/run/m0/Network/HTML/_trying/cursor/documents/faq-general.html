Ogólne pytania

Jak rozpocząć pracę z systemem?

Zapoznaj się z sekcją "Wprowadzenie" i wykonaj tutoriale. Zacznij od Tutorial 1, który poprowadzi Cię przez podstawowe kroki instalacji i konfiguracji.

Czy system jest darmowy?

Tak, system jest dostępny na licencji open source. Możesz go używać, modyfikować i rozpowszechniać zgodnie z warunkami licencji MIT.

Jakie są wymagania systemowe?

Minimalne wymagania to 4GB RAM i 2GB wolnego miejsca na dysku. System działa na Windows, Linux i macOS.

Czy system obsługuje różne bazy danych?

Tak, system obsługuje SQLite (domyślnie), PostgreSQL, MySQL i MongoDB. Możesz wybrać bazę danych odpowiednią dla Twoich potrzeb.

Jak często są wydawane aktualizacje?

Nowe wersje są wydawane co miesiąc. Aktualizacje bezpieczeństwa są wydawane w razie potrzeby.

Czy mogę współtworzyć projekt?

Tak! Wszystkie wkłady są mile widziane. Sprawdź nasze repozytorium na GitHub i przeczytaj wytyczne dla współtwórców.
