Podstawowa konfiguracja

Podstawowa konfiguracja zawiera najważniejsze ustawienia.

Parametry

  • port: Port na którym działa serwer (domyślnie 3000)
  • host: Adres hosta (domyślnie localhost)
  • debug: Tryb debugowania (domyślnie false)
  • logLevel: Poziom logowania (domyślnie info)

Przykład podstawowej konfiguracji

{
  "server": {
    "port": 3000,
    "host": "localhost"
  },
  "debug": false,
  "logLevel": "info"
}

Uruchomienie z podstawową konfiguracją

System można uruchomić z domyślnymi ustawieniami:

nasz-system start

Zmiana portu

Aby zmienić port serwera:

nasz-system start --port 8080
