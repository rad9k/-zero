Instalacja na macOS

Instalacja na macOS wymaga Xcode Command Line Tools.

Wymagania

  • macOS 10.15+
  • Xcode Command Line Tools
  • Homebrew (opcjonalnie)
  • Node.js w wersji 14+

Instalacja przez Homebrew

Najłatwiejszy sposób:

brew install nasz-system

Instalacja przez NPM

Alternatywnie:

npm install -g nasz-system

Instalacja Xcode Command Line Tools

Jeśli nie masz zainstalowanych narzędzi Xcode:

xcode-select --install

Weryfikacja instalacji

Sprawdź czy wszystko działa:

nasz-system --version
which nasz-system
