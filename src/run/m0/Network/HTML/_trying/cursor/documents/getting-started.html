Rozpoczęcie pracy

Aby rozpocząć pracę z systemem, należy wykonać kilka prostych kroków.

Wymagania

  • Node.js w wersji 14 lub nowszej
  • NPM lub Yarn
  • Przeglądarka internetowa
  • Edytor kodu (opcjonalnie)

Kroki instalacji

  1. Pobierz najnowszą wersję systemu
  2. Rozpakuj archiwum do wybranej lokalizacji
  3. Otwórz terminal w katalogu z systemem
  4. Uruchom komendę instalacyjną
  5. Skonfiguruj podstawowe ustawienia

Pierwsze uruchomienie

Po instalacji system będzie dostępny pod adresem http://localhost:3000

Konfiguracja początkowa

Przy pierwszym uruchomieniu system poprowadzi Cię przez proces konfiguracji:

  • Utworzenie konta administratora
  • Konfiguracja bazy danych
  • Ustawienie podstawowych parametrów
  • Import przykładowych danych
