Model Product

Model reprezentujący produkt w systemie.

Właściwości

  • id (number): Unikalny identyfikator
  • name (string): Nazwa produktu
  • description (string): Opis produktu
  • price (number): Cena produktu
  • category (string): Kategoria produktu
  • inStock (boolean): Czy produkt jest dostępny
  • createdAt (Date): Data utworzenia
  • updatedAt (Date): Data ostatniej aktualizacji

Walidacja

  • name: Wymagane, min. 1 znak, max. 200 znaków
  • description: Opcjonalne, max. 1000 znaków
  • price: Wymagane, liczba dodatnia
  • category: Opcjonalne, max. 100 znaków
  • inStock: Opcjonalne, domyślnie true

Metody

  • validate(): Sprawdza poprawność danych
  • isAvailable(): Sprawdza czy produkt jest dostępny
  • updateStock(): Aktualizuje stan magazynowy
  • getFormattedPrice(): Zwraca sformatowaną cenę

Przykład użycia

const product = new Product({
  name: "Laptop Dell XPS 13",
  description: "Nowoczesny laptop biznesowy",
  price: 3999.99,
  category: "Electronics",
  inStock: true
});

await product.save();
