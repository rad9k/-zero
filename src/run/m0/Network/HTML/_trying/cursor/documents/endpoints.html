Endpointy

Dostępne endpointy API:

Użytkownicy

  • GET /api/users - Pobiera listę użytkowników
  • POST /api/users - Tworzy nowego użytkownika
  • GET /api/users/:id - Pobiera użytkownika po ID
  • PUT /api/users/:id - Aktualizuje użytkownika
  • DELETE /api/users/:id - Usuwa użytkownika

Autoryzacja

  • POST /api/auth/login - Logowanie
  • POST /api/auth/logout - Wylogowanie
  • POST /api/auth/refresh - Odświeżenie tokenu
  • POST /api/auth/register - Rejestracja

Dokumenty

  • GET /api/documents - Pobiera listę dokumentów
  • POST /api/documents - Tworzy nowy dokument
  • GET /api/documents/:id - Pobiera dokument po ID
  • PUT /api/documents/:id - Aktualizuje dokument
  • DELETE /api/documents/:id - Usuwa dokument

System

  • GET /api/health - Status systemu
  • GET /api/version - Wersja systemu
  • GET /api/config - Konfiguracja systemu
