Drugi tutorial

Ten tutorial skupia się na zaawansowanych funkcjach systemu.

Tematy

  • Konfiguracja bazy danych
  • Tworzenie własnych endpointów
  • Zarządzanie uprawnieniami
  • Optymalizacja wydajności

Wymagany czas

Około 60 minut

Wymagania wstępne

  • Ukończenie Tutorial 1
  • Podstawowa znajomość JavaScript
  • Znajomość SQL (opcjonalnie)

Krok 1: Konfiguracja bazy danych

Skonfiguruj PostgreSQL zamiast domyślnego SQLite:

{
  "database": {
    "type": "postgresql",
    "host": "localhost",
    "port": 5432,
    "name": "nasz_system",
    "user": "admin",
    "password": "secret"
  }
}

Krok 2: Własne endpointy

Utwórz własny endpoint API:

app.get('/api/custom', (req, res) => {
  res.json({ message: 'Własny endpoint!' });
});

Krok 3: Uprawnienia

Skonfiguruj role i uprawnienia użytkowników.

Krok 4: Optymalizacja

Włącz cache i zoptymalizuj zapytania do bazy danych.
