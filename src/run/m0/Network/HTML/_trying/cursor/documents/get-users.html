GET /api/users

Pobiera listę wszystkich użytkowników.

Parametry

  • page (opcjonalny): Numer strony (domyślnie 1)
  • limit (opcjonalny): Liczba elementów na stronę (domyślnie 10)
  • search (opcjonalny): Fraza do wyszukiwania
  • sort (opcjonalny): Sortowanie (name, email, created_at)
  • order (opcjonalny): Kolejność (asc, desc)

Przykład żądania

GET /api/users?page=1&limit=20&search=jan&sort=name&order=asc

Odpowiedź

{
  "users": [
    {
      "id": 1,
      "name": "Jan Kowalski",
      "email": "jan@example.com",
      "role": "user",
      "createdAt": "2024-01-01T00:00:00Z",
      "updatedAt": "2024-01-01T00:00:00Z"
    }
  ],
  "total": 100,
  "page": 1,
  "limit": 20,
  "pages": 5
}

Kody odpowiedzi

  • 200 - Sukces
  • 401 - Brak autoryzacji
  • 403 - Brak uprawnień
