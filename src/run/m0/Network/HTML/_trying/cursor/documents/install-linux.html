Instalacja na Linux

Instalacja na systemie Linux jest najprostsza.

Wymagania

  • Ubuntu 18.04+ lub inna dystrybucja
  • Python 3.8+
  • GCC compiler
  • Node.js w wersji 14+

Instalacja przez pakiet

Wykonaj w terminalu:

sudo apt update
sudo apt install nasz-system

Instalacja przez NPM

Alternatywnie możesz użyć NPM:

sudo npm install -g nasz-system

Instalacja z kodu źródłowego

Dla zaawansowanych użytkowników:

git clone https://github.com/example/nasz-system.git
cd nasz-system
make install

Konfiguracja systemu

Po instalacji skonfiguruj system:

sudo nasz-system --configure
