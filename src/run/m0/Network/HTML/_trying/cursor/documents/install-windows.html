Instalacja na Windows

Instalacja na systemie Windows wymaga dodatkowych kroków.

Wymagania systemowe

  • Windows 10 lub nowszy
  • Administrator uprawnienia
  • Visual Studio Build Tools
  • Node.js w wersji 14+

Proces instalacji

  1. Uruchom PowerShell jako administrator
  2. Wykonaj komendę: npm install -g nasz-system
  3. Poczekaj na zakończenie instalacji
  4. Sprawdź czy instalacja się powiodła

Rozwiązywanie problemów

Jeśli napotkasz problemy z instalacją:

  • Upewnij się, że masz uprawnienia administratora
  • Sprawdź czy Node.js jest zainstalowany
  • Zaktualizuj NPM do najnowszej wersji
  • Wyczyść cache NPM: npm cache clean --force

Weryfikacja instalacji

Po instalacji sprawdź czy system działa:

nasz-system --version
