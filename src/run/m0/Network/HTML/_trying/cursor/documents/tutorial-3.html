Trzeci tutorial

Ostatni tutorial pokazuje jak rozszerzyć system o własne funkcjonalności.

Rozszerzenia

  • Tworzenie pluginów
  • Integracja z zewnętrznymi API
  • Własne modele danych
  • Niestandardowe widoki

Wymagany czas

Około 90 minut

Wymagania wstępne

  • Ukończenie Tutorial 2
  • Znajomość JavaScript ES6+
  • Podstawowa znajomość Node.js

Krok 1: Tworzenie pluginu

Utwórz własny plugin systemu:

class MyPlugin {
  constructor() {
    this.name = 'MyPlugin';
  }
  
  init(app) {
    // Inicjalizacja pluginu
  }
}

Krok 2: Integracja z API

Zintegruj system z zewnętrznym API:

const response = await fetch('https://api.external.com/data');
const data = await response.json();
