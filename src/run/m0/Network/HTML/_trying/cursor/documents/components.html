Komponenty

System składa się z następujących głównych komponentów:

Serwer HTTP

Obsługuje żądania HTTP i zarządza połączeniami. Wykorzystuje Express.js jako framework.

  • Obsługa żądań GET, POST, PUT, DELETE
  • Middleware do autoryzacji
  • Obsługa CORS
  • Kompresja odpowiedzi

Router

Kieruje żądania do odpowiednich kontrolerów na podstawie URL i metody HTTP.

  • Definicja tras (routes)
  • Middleware dla tras
  • Parametry URL
  • Walidacja żądań

Baza danych

Przechowuje dane aplikacji. Obsługuje różne typy baz danych.

  • SQLite (domyślnie)
  • PostgreSQL
  • MySQL
  • MongoDB

Cache

Przyspiesza dostęp do często używanych danych.

  • Redis (opcjonalnie)
  • Cache w pamięci
  • Automatyczne wygasanie
  • Invalidacja cache

Autoryzacja

Zarządza dostępem użytkowników do zasobów systemu.

  • JWT tokens
  • Role i uprawnienia
  • Session management
  • OAuth 2.0 (opcjonalnie)
