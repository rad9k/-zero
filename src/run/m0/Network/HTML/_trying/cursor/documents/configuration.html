Konfiguracja

Po instalacji należy skonfigurować system według własnych potrzeb.

Plik konfiguracyjny

Główny plik konfiguracyjny znajduje się w /etc/nasz-system/config.json

Podstawowe ustawienia

Minimalna konfiguracja wymaga ustawienia:

  • Port serwera
  • Adres hosta
  • Tryb debugowania
  • Ścieżka do bazy danych

Przykład konfiguracji

{
  "server": {
    "port": 3000,
    "host": "localhost"
  },
  "database": {
    "path": "/var/lib/nasz-system/db.sqlite"
  },
  "debug": false,
  "logLevel": "info"
}

Konfiguracja przez interfejs

Możesz skonfigurować system przez interfejs webowy:

  1. Uruchom system
  2. Otwórz przeglądarkę
  3. Przejdź do http://localhost:3000/admin
  4. Wypełnij formularz konfiguracyjny
