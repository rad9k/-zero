2. Instalacja i Konfiguracja

Proces instalacji został zaprojektowany tak, aby był jak najprostszy. Postępuj zgodnie z poniższymi krokami, aby poprawnie zainstalować i skonfigurować system na swoim środowisku.

  1. Pobierz najnowszą wersję instalatora.
  2. Uruchom instalator i postępuj zgodnie z instrukcjami na ekranie.
  3. Po zakończeniu instalacji, uruchom aplikację w celu dokonania wstępnej konfiguracji.
