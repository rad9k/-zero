Tutoriale

Kolekcja tutoriali pomagających w nauce korzystania z systemu.

Tutoriale są uporządkowane od najprostszych do najbardziej zaawansowanych.

Dla początkujących

Tutorial 1: Pierwsze kroki

W tym tutorialu nauczysz się podstaw korzystania z systemu.

  • Czas trwania: 30 minut
  • Poziom: Początkujący
  • Wymagania: Brak

Dla średnio zaawansowanych

Tutorial 2: Konfiguracja zaawansowana

Ten tutorial skupia się na zaawansowanych funkcjach systemu.

  • Czas trwania: 60 minut
  • Poziom: Średnio zaawansowany
  • Wymagania: Ukończenie Tutorial 1

Dla zaawansowanych

Tutorial 3: Rozszerzanie systemu

Ostatni tutorial pokazuje jak rozszerzyć system o własne funkcjonalności.

  • Czas trwania: 90 minut
  • Poziom: Zaawansowany
  • Wymagania: Ukończenie Tutorial 2

Materiały dodatkowe

Każdy tutorial zawiera:

  • Szczegółowe instrukcje krok po kroku
  • Przykłady kodu
  • Ćwiczenia praktyczne
  • Test sprawdzający wiedzę
