Model User

Model reprezentujący użytkownika w systemie.

Właściwości

  • id (number): Unikalny identyfikator
  • name (string): Imię i nazwisko użytkownika
  • email (string): Adres email (unikalny)
  • password (string): Zahashowane hasło
  • role (string): Rola użytkownika (user, admin)
  • createdAt (Date): Data utworzenia
  • updatedAt (Date): Data ostatniej aktualizacji

Walidacja

  • name: Wymagane, min. 2 znaki, max. 100 znaków
  • email: Wymagane, poprawny format email, unikalne
  • password: Wymagane, min. 8 znaków
  • role: Opcjonalne, domyślnie "user"

Metody

  • validate(): Sprawdza poprawność danych
  • hashPassword(): Hashuje hasło
  • comparePassword(): Porównuje hasło z hashem
  • toJSON(): Zwraca dane bez hasła

Przykład użycia

const user = new User({
  name: "Jan Kowalski",
  email: "jan@example.com",
  password: "securePassword123",
  role: "user"
});

await user.save();
