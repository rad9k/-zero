Instalacja

Instalacja systemu jest bardzo prosta i zajmuje tylko kilka minut.

Kroki instalacji

  1. Pobierz najnowszą wersję systemu
  2. Rozpakuj archiwum do wybranej lokalizacji
  3. Otwórz terminal w katalogu z systemem
  4. Uruchom komendę instalacyjną
  5. Skonfiguruj podstawowe ustawienia

Wymagania systemowe

  • Node.js w wersji 14 lub nowszej
  • NPM lub Yarn
  • Przeglądarka internetowa
  • Edytor kodu (opcjonalnie)

Automatyczna instalacja

System oferuje automatyczny skrypt instalacyjny:

npm install -g nasz-system

Instalacja z kodu źródłowego

Jeśli chcesz zainstalować z kodu źródłowego:

git clone https://github.com/example/nasz-system.git
cd nasz-system
npm install
npm run build
