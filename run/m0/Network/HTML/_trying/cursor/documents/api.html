API Reference

Dokumentacja API zawiera szczegółowe informacje o wszystkich dostępnych endpointach.

API jest RESTful i zwraca dane w formacie JSON.

Podstawowe informacje

  • Base URL: https://api.example.com/v1
  • Format danych: JSON
  • Autoryzacja: Bearer Token
  • Rate limiting: 1000 requestów na godzinę

Autoryzacja

Wszystkie żądania do API wymagają autoryzacji. Użyj nagłówka Authorization:

Authorization: Bearer YOUR_API_TOKEN

Kody odpowiedzi

  • 200 - Sukces
  • 201 - Utworzono
  • 400 - Błędne żądanie
  • 401 - Brak autoryzacji
  • 404 - Nie znaleziono
  • 500 - Błąd serwera

Przykład użycia

curl -X GET \
  https://api.example.com/v1/users \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -H 'Content-Type: application/json'
