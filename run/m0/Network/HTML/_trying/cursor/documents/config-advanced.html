Zaawansowana konfiguracja

Zaawansowana konfiguracja pozwala na dostosowanie wszystkich aspektów systemu.

Opcje zaawansowane

  • Konfiguracja bazy danych
  • Ustawienia bezpieczeństwa
  • Konfiguracja cache
  • Ustawienia logowania
  • Konfiguracja SSL/TLS
  • Ustawienia proxy

Przykład zaawansowanej konfiguracji

{
  "server": {
    "port": 3000,
    "host": "localhost",
    "ssl": {
      "enabled": true,
      "cert": "/path/to/cert.pem",
      "key": "/path/to/key.pem"
    }
  },
  "database": {
    "type": "postgresql",
    "host": "localhost",
    "port": 5432,
    "name": "nasz_system",
    "user": "admin",
    "password": "secret"
  },
  "security": {
    "jwtSecret": "your-secret-key",
    "bcryptRounds": 12,
    "sessionTimeout": 3600
  },
  "cache": {
    "enabled": true,
    "type": "redis",
    "host": "localhost",
    "port": 6379
  },
  "logging": {
    "level": "debug",
    "file": "/var/log/nasz-system.log",
    "maxSize": "10MB",
    "maxFiles": 5
  }
}

Konfiguracja bazy danych

System obsługuje różne typy baz danych:

  • SQLite (domyślnie)
  • PostgreSQL
  • MySQL
  • MongoDB
