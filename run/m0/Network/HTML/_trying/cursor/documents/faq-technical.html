Pytania techniczne

Jak zmienić port serwera?

Edytuj plik konfiguracyjny i zmień parametr "port" w sekcji "server". Możesz też użyć flagi --port przy uruchamianiu:

nasz-system start --port 8080

Jak włączyć tryb debugowania?

Ustaw parametr "debug" na true w konfiguracji lub użyj flagi --debug:

nasz-system start --debug

Jak zresetować hasło administratora?

Użyj komendy:

npm run reset-admin

Jak skonfigurować SSL/TLS?

Dodaj sekcję SSL do konfiguracji:

{
  "server": {
    "ssl": {
      "enabled": true,
      "cert": "/path/to/cert.pem",
      "key": "/path/to/key.pem"
    }
  }
}

Jak włączyć cache Redis?

Dodaj konfigurację cache do pliku konfiguracyjnego:

{
  "cache": {
    "enabled": true,
    "type": "redis",
    "host": "localhost",
    "port": 6379
  }
}

Jak zrobić backup bazy danych?

Użyj komendy backup:

nasz-system backup --output backup.sql

Jak zaktualizować system?

Zaktualizuj system używając NPM:

npm update -g nasz-system

Jak sprawdzić logi systemu?

Logi są zapisywane w pliku określonym w konfiguracji. Możesz też użyć:

nasz-system logs
