## Platform instances

The - zero requirements are:
	- **desktop** version requires Windows _(x64, arm64)_
	- **console** version works on: 
		- Windows _(x64, arm64)_
		- macOS _(x64, arm64)_
		- Linux _(x64, arm64)_
